# OllamaCopilot

OllamaCopilot is a local AI assistant built using Ollama and Python.

## Features

- AI Chat
- Memory System
- Document Chat (RAG)
- Resume Analyzer
- Interview Coach
- GitHub Repository Analyzer

## Tech Stack

- Ollama
- Python
- Streamlit
- ChromaDB
- SQLite
